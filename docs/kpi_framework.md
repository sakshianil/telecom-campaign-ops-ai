# KPI Framework

## Pilot KPIs

The pilot should measure whether the campaign workflow is ready to scale, not only whether sales happen immediately.

## Core Metrics

| KPI | Meaning | Why it matters |
| --- | --- | --- |
| Leads reviewed | Number of records scored and prepared | Shows data readiness |
| High-priority queue | Leads needing fast contact | Guides agent allocation |
| Contact rate | Successful conversations divided by attempts | Measures list quality |
| Callback rate | Customers who agree to a next step | Measures interest |
| Conversion rate | Upgrades, renewals, or saves | Measures campaign outcome |
| Objection rate | Most common blocker categories | Improves scripts |
| CRM completeness | Records with full notes | Protects operational quality |

## Daily Manager Review

- Which segment converted or responded best?
- Which objections appeared repeatedly?
- Which agents need coaching?
- Which offers need clearer wording?
- Which data fields were missing or unreliable?

## Scale Criteria

Scale beyond 2-5 agents only when:

- CRM notes are consistently complete.
- Top objections are understood.
- Scripts are stable enough for new agents.
- The manager can see daily performance clearly.
- Follow-up ownership is not falling through gaps.
