# Two-Week Pilot Plan

## Week 1: Build and Test

### Day 1

- Confirm target telecom campaign.
- Define customer segments and success metrics.
- Prepare synthetic or client-approved sample data.

### Day 2

- Build lead tracker and segmentation fields.
- Draft campaign scripts.
- Confirm CRM update requirements.

### Day 3

- Train 2-5 pilot agents.
- Run mock calls.
- Revise opening lines and objection handling.

### Day 4

- Start small calling block.
- Review CRM notes after every block.
- Identify missing fields and confusing script sections.

### Day 5

- Produce first manager report.
- Adjust prioritization and channel strategy.

## Week 2: Improve and Scale Decision

### Days 6-8

- Continue outreach.
- Review objections daily.
- Test follow-up sequences.

### Days 9-10

- Compare performance by segment and channel.
- Decide whether to scale, pause, or redesign the campaign.

## Final Deliverables

- Segmented campaign dataset.
- Agent scripts and onboarding guide.
- Follow-up workflow.
- KPI dashboard.
- Manager report.
- Scale recommendation.
